 Multivendor
 A multi-vendor platform is an online marketplace where multiple sellers can register, create their own stores, and sell products through a single system. Instead of one seller controlling all products, each vendor manages their own listings, pricing, and orders while customers can browse and purchase from different sellers in one place.

The purpose of this project is to build a scalable e-commerce marketplace similar to platforms like Amazon or Daraz. It helps understand how real-world multi-seller systems work, including vendor management, product handling, and order processing. It is mainly used to gain practical experience in building complex e-commerce architectures where multiple users interact within a shared platform.
