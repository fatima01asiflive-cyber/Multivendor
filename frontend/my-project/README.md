# Multivendor Platform - Frontend

A modern React + Vite + TypeScript frontend for the multivendor ecommerce platform.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build
